# 🚀 Public API Explorer

A premium, interactive playground for exploring public APIs without any setup or keys. Built with modern web technologies, glassmorphism, and smooth animations.

![Public API Explorer Hero Image](https://via.placeholder.com/1200x600/080814/ffffff?text=Public+API+Explorer)

## ✨ Features

- **Live Dog Finder**: Fetches random dog images and identifies breeds from the API path.
- **Joke Generator**: Instant setups and punchlines for quick laughs.
- **Random User Profile**: Generates complete user data with avatars, contact info, and location.
- **JSONPlaceholder Posts**: Fetches and displays blog-style posts from a placeholder API.
- **Interactive UI**: Custom cursor, particle burst effects, and dynamic glassmorphism cards.
- **Fully Responsive**: Optimized for both desktop and mobile devices.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Custom design system, CSS Grid, and high-quality animations.
- **Modern JavaScript (ES6+)**: Async/Await fetch logic and UI orchestration.
- **Fonts**: Clash Display and Instrument Sans for a premium typographic feel.

## 🚀 Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/divyansha12/Public-API-Explorer.git
    ```
2.  Open `index.html` in your browser.
    - *No dependencies or local server required!*

## 📁 Project Structure

- `index.html`: Core structure and API card containers.
- `style.css`: Design system, animations, and responsive layout.
- `app.js`: API fetching logic and interactive UI behaviors.

## 🌐 APIs Used

- [Dog API](https://dog.ceo/dog-api/)
- [Official Joke API](https://github.com/15Dkatz/official_joke_api)
- [Random User Generator](https://randomuser.me/)
- [JSONPlaceholder](https://jsonplaceholder.typicode.com/)
